# Crack-Detection-CNN

## 📸 Cracked vs Non-Cracked Image Classification

Welcome to the **Cracked vs Non-Cracked Image Classification** project! This project utilizes a Convolutional Neural Network (CNN) to classify images as either cracked or non-cracked, providing a reliable solution for defect detection. The project aims to contribute to predictive maintenance and quality control systems in industries such as construction, manufacturing, and infrastructure monitoring.
